Accident alerts

When a municipality is on accident alert, police should be notified of an accident only if it is blocking traffic, medical attention is required, drugs or alcohol are involved, a driver flees the scene, a driver doesn't have a valid license or insurance, or public property is involved. Otherwise, drivers should exchange license, registration and insurance information and file an accident report as soon as possible.

For all municipalities, an online accident report can be filed with the Colorado State Patrol: Colorado Online Accident Report.
